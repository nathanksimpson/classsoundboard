How to Use

English Guide

Use this soundboard to quickly play, organize, and manage sounds.

  • Add sounds with Add Sound or Add Web Sound.
  • Edit a tile with the pencil icon (or right-click on desktop).
  • Assign hotkeys like Q, Shift+A, or Shift+.
  • Use Hotkeys only to show only sounds with hotkeys.
  • Use Reorder to drag sounds/categories into new positions.
  • Use Manage All Sounds for bulk editing and saving.

Useful Hotkeys for default sounds

  • 3 - Mario Kart 3-2-1 Go
  • X - Wrong answer buzzer
  • R - Great!
  • A - Anime Wow
  • 9 - Sad music
  • P - Perfect
  • M - Stop, wait a minute
  • V - Victory sound
  • 1 - One more Time

한국어 안내

이 사운드보드는 사운드를 빠르게 재생하고 정리/관리할 수 있도록 만들어졌습니다.

  • Add Sound 또는 Add Web Sound로 사운드를 추가하세요.
  • 타일의 연필 아이콘(PC는 우클릭)으로 편집할 수 있습니다.
  • Q, Shift+A, Shift+. 같은 단축키를 지정할 수 있습니다.
  • Hotkeys only로 단축키가 있는 사운드만 볼 수 있습니다.
  • Reorder에서 드래그하여 사운드/카테고리 순서를 바꿀 수 있습니다.
  • Manage All Sounds에서 전체 일괄 편집 후 저장할 수 있습니다.

Manage All Sounds (Bulk Editor)

Edit all sounds in one scrollable list. Remove a sound by checking “Delete”. Press Enter to save and continue editing. Hotkeys must be unique.
